---
title: PDA utilities
description: Functions for deriving program-derived addresses
---

All PDA utilities use `PublicKey.findProgramAddressSync` to deterministically derive addresses.

## State and vault PDAs

### getStatePda

Derives the state account PDA for a vault.

```typescript
function getStatePda(
  initKey: Uint8Array | number[],
  owner: PublicKey,
  programId: PublicKey,
): PublicKey
```

<ParamField path="initKey" type="Uint8Array | number[]" required>
  Initialization key (unique identifier for the vault)
</ParamField>

<ParamField path="owner" type="PublicKey" required>
  Vault owner public key
</ParamField>

<ParamField path="programId" type="PublicKey" required>
  GLAM protocol program ID
</ParamField>

<ResponseField name="return" type="PublicKey">
  State account PDA
</ResponseField>

### getVaultPda

Derives the vault PDA for a state account.

```typescript
function getVaultPda(
  statePda: PublicKey,
  programId: PublicKey,
): PublicKey
```

<ParamField path="statePda" type="PublicKey" required>
  State account PDA
</ParamField>

<ParamField path="programId" type="PublicKey" required>
  GLAM protocol program ID
</ParamField>

<ResponseField name="return" type="PublicKey">
  Vault PDA
</ResponseField>
