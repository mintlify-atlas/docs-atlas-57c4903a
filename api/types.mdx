## Request queue types

### RequestQueue

Queue of pending subscription/redemption requests.

<ResponseField name="data" type="PendingRequest[]" required>
  Array of pending requests
</ResponseField>

<ResponseField name="subscriptionPaused" type="boolean" required>
  Whether subscriptions are paused
</ResponseField>

<ResponseField name="redemptionPaused" type="boolean" required>
  Whether redemptions are paused
</ResponseField>

### PendingRequest

A single pending subscription or redemption request.

<ResponseField name="investor" type="PublicKey" required>
  Investor public key
</ResponseField>

<ResponseField name="amount" type="BN" required>
  Request amount
</ResponseField>

<ResponseField name="requestType" type="number" required>
  Request type (0 = subscription, 1 = redemption)
</ResponseField>

<ResponseField name="timestamp" type="BN" required>
  Request timestamp
</ResponseField>
